# Monitoring Population Density in Railway Areas

Hệ thống phát hiện, theo dõi và đếm người trong khu vực đường sắt sử dụng thị giác máy tính (YOLO + Tracking), có cảnh báo mức độ nguy hiểm theo mật độ người.

---

## 1. Requirements

- Python 3.8+
- OpenCV
- PyTorch
- Webcam hoặc video đầu vào

---

## 2. Cài đặt môi trường

### 2.1 Clone repository
```bash
git clone https://github.com/Thangleee/Monitoring-population-density-in-railway-areas.git
```
### 2.2 Cài đặt thư viện
```
pip install -r requirements.txt
```
